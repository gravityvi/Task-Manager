Task manager created for pofessors according to priority .

There are 4 types of Tasks(this are according to priority order)

1. Teaching
2. Research
3. Academics
4. Administrative

Client is provide with 4 options:
1. add a new task
2. next pending task(displays task according to its priority displayed above and time added)
3. next task completed(ask the task manager to delete next pending task as it is completed)
4. display all pending task
5. exit program

*client adds his task with giving it's types and a small description
*he completes his task according to the priority of task which is decided according to it's type and   the time(task enetered first should be done first)
*In displaying task all tasks are displayed under the tittle a of their type
